## Generate a dir with files exec

<h4 style="color: red;">Just works on linux distributions</h4>

#### Step 1 | (Optional if can't execute)
```
chmod +X ./makebuild.sh
```

#### Step 2
```
./makebuild.sh
```

#### Step 3
###### Exec server
```
cd build && ./server
```

#### Step 4
###### Exec client
```
cd build && ./client
```
